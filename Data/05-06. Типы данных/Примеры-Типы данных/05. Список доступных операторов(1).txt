Перечень некоторых основных операторов языков C/C++:

=   - присвоить, A = 10;
+   - сложение,  B = A + 20; теперь в B находится значение 20  (10 + 20).
-   - вычитание, B = A - 5;  теперь в B находится значение 15  (10 - 5).
*   - умножение, B = A * 2;  теперь в B находится значение 15  (10 * 2).
/   - деление,   B = A / 2;  теперь в B находится значение  5  (10 / 2).
%   - целочисленное деление с остатком, вычисление остатка от деления целых чисел,
                 B = A % 4;  теперь в B находится значение  2  (остаток от деления 10 на 4).

++  - инкремент, увеличение значения на 1, бывает постфиксный и префиксный:
       постфиксный: A = 10;
                    B = A++; теперь в A равен 11 (A = A + 1), а B = 10; т.е. сначала присваивание, потом инкремент.

       префиксный:  A = 10;
                    B = ++A; теперь в A равен 11 (A = A + 1), и B = 11; т.е. сначала инкремент, потом присваивание.

--  - декремент, уменьшение значения на 1, бывает постфиксный и префиксный:
       постфиксный: A = 10;
                    B = A--; теперь в A равен 9 (A = A - 1), а B = 10; т.е. сначала присваивание, потом декремент.

       префиксный:  A = 10;
                    B = --A; теперь в A равен 9 (A = A - 1), и B = 9; т.е. сначала декремент, потом присваивание.
