Домашнее задание
Тема: Функции


1. *Напишите функцию, которая принимает размеры прямоугольника и выводит на экран прямоугольник из звездочек.

2. *Напишите функцию, которая определяет количество элементов передаваемого
массива, которые больше определенного числа, это число тоже передается в
функцию.

3. **Дан одномерный массив целых чисел. Напишите функцию, которая осуществляет нахождения искомого значения в массиве.
Функция возвращает позицию вхождения искомого значения или -1, если такого значения в массиве нет.
Проиллюстрируйте работу функции примером.

4. **Напишите функцию, которая принимает три одномерных массивов целых чисел.
Выполнить поэлементное сложение элементов первых двух массивов, и результат занести в третий массив.

5. ***Число называется совершенным, если сумма всех его делителей равна ему самому.
Напишите функцию поиска таких чисел во введенном интервале.


Дополнительные задания

6. **Напишите функцию, которая принимает вещественное число и количество знаков и возвращает округленное число до указанного знака. 

7. ***Написать функцию конкатенации (объединения) двух массивов в третий массив
void Concat(int A[], int B[], int C[], int SizeA, int SizeB),
которая последовательно копирует массивы A и B размеров SizeA и SizeB в третий массив размером SizeA + SizeB.
